# Portfolio

This is my web portfolio made with Vue. Feel free to use it as a template! All you need to do is change the contents of the "content" folder JSONs and some of the actual components and you're good to go.

## Project Setup

```sh
yarn
```

### Compile and Hot-Reload for Development

```sh
yarn dev
```

### Type-Check, Compile and Minify for Production

```sh
yarn build
```

### Lint with [ESLint](https://eslint.org/)

```sh
yarn lint
```
